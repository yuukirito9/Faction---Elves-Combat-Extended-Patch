# Faction - Elves Combat Extended Patch

Combat Extended compatibility patch for Faction - Elves.

## Features

- Combat Extended weapon support
- Great Long Bow ammunition
- Armor adjustments
- CE backpacks
- Updated pawn loadouts

## Requirements

- Harmony
- Combat Extended
- Faction - Elves

## Installation

Subscribe on Steam Workshop or place the mod after:

- Combat Extended
- Faction - Elves

## Compatibility

RimWorld 1.6
